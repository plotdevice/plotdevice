NodeBox Libraries

photobot.canvas()

Syntax

variable = canvas(width, height)

Description

Returns an empty Canvas object, with the given width and height in pixels. A canvas is an empty Photoshop document, where layers can be placed and manipulated. The canvas background is transparent, but a background color could be set using the canvas fill() function, for example.

Example

photobot = ximport("photobot")
canvas = photobot.canvas(200, 200)
