photobot.Layer.color()

Syntax

canvas.layers[i].color()

Description

Sets the blend mode of layer i to color. Replaces the hue and saturation of the bottom layer with the hue and saturation of the hued layer, while retaining the bottom layerâ€™s brightness.

Colorizing works significantly slower than other blend modes, because it is a blend mode native to PhotoBot and not of the underlying Python Imaging Library.

Example

photobot = ximport("photobot")
canvas = photobot.canvas(100,100)
canvas.layer("robot.jpg")
i = canvas.fill((185,255,0))
canvas.layers[i].color()
